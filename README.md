
# SmartMart Retail Analytics Dashboard

An end-to-end retail analytics project using Snowflake and Streamlit.

## Architecture
AWS S3 → Snowflake Data Warehouse → Streamlit Dashboard

## Features
- Region-wise revenue analysis
- Monthly sales trends
- Category and product performance
- Online vs Offline channel comparison
- Interactive filters and data explorer

## Tech Stack
- Snowflake
- Python
- Streamlit
- Pandas
- Snowflake Connector
